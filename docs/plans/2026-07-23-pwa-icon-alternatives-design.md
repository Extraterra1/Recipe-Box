# PWA Icon Alternatives Design

## Objective

Create three clean, flat alternatives to the current Recipe Box PWA icon. Each alternative must remain recognizable at small launcher and favicon sizes while feeling consistent with the app's warm cream and burgundy visual language.

## Shared Visual System

- Square 512 x 512 composition with a generously rounded app-icon background.
- Warm cream background with burgundy as the primary mark color.
- Flat, vector-like construction with crisp edges and a strong silhouette.
- No text, gradients, shadows, photorealism, borders, or fine decorative detail.
- Generous safe-area padding so each symbol works with PWA maskable cropping.

## Alternatives

### 1. Cookie Protagonist

A large centered chocolate-chip cookie with one clearly readable bite. The cookie is warm golden brown, outlined or accented in burgundy, with a small number of oversized chocolate chips. It should feel friendly without becoming childish or character-like.

### 2. Recipe Card Box

A simplified open recipe box with two visible cards. The box and cards use a small number of geometric shapes, with one short line or tab detail suggesting stored recipes. The silhouette should communicate both collection and cooking without resembling a generic filing cabinet.

### 3. Whisk Monogram

A minimal upright whisk whose wire loops subtly suggest the letter R. The monogram should read first as a kitchen utensil and only reveal the letterform secondarily. Its shape should be bold enough to survive favicon reduction.

## Deliverables and Validation

Generate each direction independently and save all three in a new project asset folder without replacing the active icon. Inspect the full-size results and compare reduced previews at 180, 64, and 32 pixels. Reject outputs with accidental text, excessive detail, weak contrast, clipped artwork, or inconsistent styling.
